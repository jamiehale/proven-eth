Proven Contracts

This project uses the [Truffle](http://truffleframework.com/) framework and [openzeppelin](https://openzeppelin.org/) smart contracts.

Commands:

  Compile:        truffle compile
  Migrate:        truffle migrate
  Test contracts: truffle test
