Proven Contracts

# Prerequisites
`npm install`

# Building
`npm run build`

... produces ABI, binary, and assembly in the `./output` directory.

# Deployment
Requires a local Ethereum node (such as [Parity](https://github.com/ethcore/parity) running on port 8545:
`parity --geth`

`npm run deploy`

# Testing
`firefox test/index.html`
